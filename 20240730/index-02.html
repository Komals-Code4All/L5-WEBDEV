So, finally, we add Media Query in the CSS.

In Mobile view every column takes 100% of the width, giving the visual effect of stacked columns.

As the screen gets larger columns become 50%, 33% or 20%. Try resizing, with breakpoints set at 480 pixel and 768 pixel.

If using ODD number of columns, you will have to consider what to do with gaps on the page. A quick fix could be to use EVEN number of columns.

NB: Use inspect to drop down to 480px width.
