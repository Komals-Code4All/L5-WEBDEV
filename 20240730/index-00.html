My First site

In this example, we are building a basic layout template.

This uses the idea of rows and columns, where a row comprises of 1 or more columns.

We could use Bootstrap, CSS-Grid or CSS-Flexbox. However, this example is based on the simple idea of using percentage widths to control how columns behave across the page.

NB: The columns expand but don't stack up. That is in the next example
